Projects > Dimensions API LAB     #data   #python   #research-analytics   #website

A collection of open source Jupyter notebooks showing how to carry out common research data analytics tasks using the Dimensions Analytics API.

Key features of this project:

  • Python notebooks demonstrating applications of the Dimensions API for research data analytics
  • All notebooks developed so that they work out-of-the-box with both standard Jupyter and Google Colab environments
  • API Lab website generated dynamically from a collection of notebooks, thanks to NBSphinx

    • Role:   Design and development of the notebooks and the website publishing workflow.
